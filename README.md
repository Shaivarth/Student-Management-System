# 🎓 Student Management System (Java)

## 📌 Description
This is a simple console-based Java application to manage student records.  
It allows users to add, view, search, and delete students.

## 🚀 Features
- Add new student
- View all students
- Search student by ID
- Delete student

## 🛠️ Technologies Used
- Java
- OOP Concepts
- ArrayList
- Scanner
